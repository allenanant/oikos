# Variant 02 — Full-Bleed Photographic

**Direction:** A single, breathtaking biophilic photograph fills the entire viewport. The headline is overlaid in cream Garamond italic, bottom-left, with restrained but rich typography.

**Signature move:**
- Full-bleed Unsplash photograph: a sunlit room with cascading plants, or a single dramatic Monstera leaf with morning light. Find one with warm tones (golden-hour light, terracotta walls, or earthy textures). Search Unsplash for: `biophilic interior morning light`, `monstera sunlight`, `linen botanical room`.
- Subtle dark-cream gradient at the bottom 35% of the photo for headline legibility
- Headline anchored bottom-left with about 8% padding
- Hidden top-right: a small terracotta circle with the year `2024` and below it the word `Delhi` set in Inter all-caps tracking — quietly editorial

**Inspiration tone:** Cereal magazine cover, Aesop store photography, Nordic Hygge meets Mediterranean warmth

**Constraints:**
- Use a real Unsplash URL (one of these or find better via web search):
  - `https://images.unsplash.com/photo-1545241047-6083a3684587?w=2400&q=80&auto=format&fit=crop`
  - `https://images.unsplash.com/photo-1497366216548-37526070297c?w=2400&q=80&auto=format&fit=crop`
  - Or search for similar
- The photo CANNOT clash with the brand palette — if it has cool blue/gray tones, apply a `filter: sepia(0.15) saturate(1.1)` overlay to warm it
- Cycler optional but if used, only the verb cycles (sustains / nurtures / steadies / shelters) — keep "you." static
